# Archive Rooms

Archving a room marks it as read only and then removes it from the channel list on the left.

## Requirements

| Logged In | Permission     | Setting |
| --------- | -------------- | ------- |
| Yes       | `archive-room` | _none_  |

## Payload

1. String - the id of the room to archive

## Example Call

```json
{
    "msg": "method",
    "method": "archiveRoom",
    "id": "97",
    "params": [
        "roomId"
    ]
}
```

## Example Response

```json
{
    "msg": "result",
    "id": "97"
}
```
