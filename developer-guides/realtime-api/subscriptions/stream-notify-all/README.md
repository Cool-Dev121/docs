# stream-notify-all

General user-wide stream.

Events available:

- roles-change
- updateEmojiCustom
- deleteEmojiCustom
- updateAvatar
- public-settings-changed
- permissions-changed

```json
{
    "msg": "sub",
    "id": "unique-id",
    "name": "stream-notify-all",
    "params":[
        "event",
        false
    ]
}
```
