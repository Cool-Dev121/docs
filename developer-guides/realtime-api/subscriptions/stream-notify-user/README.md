# stream-notify-user

This is the user stream.

Events available:

 - message
 - otr
 - webrtc
 - notification
 - rooms-changed
 - subscriptions-changed

```json
{
    "msg": "sub",
    "id": "unique-id",
    "name": "stream-notify-user",
    "params":[
        "user-id/event",
        false
    ]
}
```
